# Breakaway-Trivia
Hockey trivia application that delivers a personalized quiz experience based on the user's favorite NHL team selected. 

## Overview:
Breakaway Trivia is a hockey trivia application that delivers a personalized quiz experience based on the user's favorite NHL team. Users select their team and are challenged with trivia questions spanning team history, players, stats, trades, and iconic moments.
Designed for die-hard fans, the app combines team-specific content with an engaging quiz format that will make every session feel personal to the fan playing it.
